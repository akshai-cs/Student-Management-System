# Student Management System

A responsive Student Management System dashboard built using React.js and Tailwind CSS. The application provides a modern admin interface for managing student records and viewing dashboard analytics.

## Features

- Student records management
- Dashboard analytics cards
- Student search functionality
- Responsive admin dashboard UI
- Sidebar navigation
- Modern card and table layout

## Tech Stack

- React.js
- Tailwind CSS
- JavaScript
- Vite

## Future Enhancements

- Backend integration using Node.js and Express.js
- MongoDB database connectivity
- Authentication system
- Attendance management
- Role-based access

## Installation

```bash
npm install
npm run dev
```
